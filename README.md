# How to Use This Framework

<img width="2068" height="944" alt="image" src="https://github.com/user-attachments/assets/8cc3dc56-d17a-4c82-affc-3aa80b3e38d3" />

Follow the steps below to set up the benchmark environment and run the *vlm-psp* extension inside NVIDIA Omniverse.

---

## **1) Install Omniverse**

Download and install Omniverse from the official repository:
**[https://github.com/NVIDIA-Omniverse](https://github.com/NVIDIA-Omniverse)**

---

## **2) Download the Scene**

Clone or download the scene files provided in this repository.

---

## **3) Download the `vlm-psp` Extension**

Get the extension folder from this repository and place it inside your Omniverse `extensions` directory.

---

## **4) Build Omniverse and Enable the Extension**

Rebuild Omniverse (if required by your setup) and activate the `vlm-psp` extension in the Extension Manager.

---

## **5) Load the Scene**

Open Omniverse and load the provided USD scene.

---

## **6) Configure VLM API Settings**

Open the extension settings and provide any required API keys or access tokens.

<img width="428" height="212" alt="image" src="https://github.com/user-attachments/assets/9ade4fc9-93ce-4454-ad87-8c47c8a2beed" />

---

## **7) Start Exploring**

Use the Intelligent Assistant panel to ask questions and test navigation.
